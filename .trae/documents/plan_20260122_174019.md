# 安装新添加的依赖

## 需要安装的依赖
我们在以下文件中添加了新的依赖：
- [packages/db/package.json](file:///d:/project/imagePrompt/packages/db/package.json)：添加了 `pg`、`prisma-kysely`、`@types/pg`
- [packages/auth/package.json](file:///d:/project/imagePrompt/packages/auth/package.json)：添加了 `kysely`、`pg`

## 安装命令
在项目根目录运行：
```bash
bun install
```

或者使用 npm：
```bash
npm install
```

## 说明
- `bun install` 会自动安装所有工作区（workspace）的依赖
- 包括新添加的 `kysely` 和 `pg` 包
- 安装完成后，数据库连接应该可以正常工作
