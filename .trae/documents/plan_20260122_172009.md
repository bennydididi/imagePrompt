# 修复 Middleware 导出错误

## 问题
`middleware.ts` 文件使用了错误的导入语法：
- 当前：`import { middleware } from "./utils/nextauth"`
- 但 `nextauth.ts` 使用的是 `export default`

## 解决方案
修改 [apps/nextjs/src/middleware.ts](file:///d:/project/imagePrompt/apps/nextjs/src/middleware.ts) 的导入语句，从命名导入改为默认导入：

```typescript
// 修改前
import { middleware } from "./utils/nextauth";

// 修改后
import middleware from "./utils/nextauth";
```

## 说明
当文件使用 `export default` 导出时，应该使用默认导入（不带花括号），而不是命名导入（带花括号）。
