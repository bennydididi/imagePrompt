# 修复 Kysely 配置错误

## 问题原因
Kysely 的 `dialect` 参数需要是一个 `PostgresDialect` 实例，而不是一个包含 pool 的对象。

**错误配置：**
```typescript
const pool = new Pool({ connectionString: process.env.POSTGRES_URL });
export const db = new Kysely<Database>({ dialect: { pool } });
```

**正确配置：**
```typescript
import { Kysely, PostgresDialect } from 'kysely';
import { Pool } from 'pg';

const pool = new Pool({ connectionString: process.env.POSTGRES_URL });
export const db = new Kysely<Database>({ 
  dialect: new PostgresDialect({ pool }) 
});
```

## 解决方案
修改 [packages/db/index.ts](file:///d:/project/imagePrompt/packages/db/index.ts) 和 [packages/auth/db.ts](file:///d:/project/imagePrompt/packages/auth/db.ts)，使用正确的 Kysely 配置方式。
